## Install

```shell

chmod +x rainbow.sh
```
or
```shell
wget https://raw.githubusercontent.com/ItsJimi/rainbow.sh/main/rainbow.sh
chmod +x rainbow.sh
```
or
```shell
git clone https://github.com/ItsJimi/rainbow.sh
```

## Launch
```shell
./rainbow.sh
```
